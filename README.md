# AI IT Support Assistant

A Python-based IT support chatbot designed to help students resolve common technical issues at a community college.

## Features
- Rule-based routing system with scoring improvements
- Content-driven responses (D2L, Wi-Fi, password reset, etc.)
- SQLite logging of user interactions
- Reporting and export tools for analysis

## Tech Stack
- Python (Flask)
- SQLite
- Custom routing logic

## Status
Currently in prototype phase. Recently upgraded routing from keyword matching to a scored matching system with evaluation tests.
