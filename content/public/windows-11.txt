  WINDOWS 11 TRAINING AND UPGRADE

  What this is:

  - Information about Windows 11 training resources, updating mapped drives in Windows, and the Windows 11 upgrade
  process.
  - Training is available through LinkedIn Learning for staff, faculty, instructors, and students.
  - Users sign in to LinkedIn Learning with their CCA email, then use their S# and MyCCA password through single sign-
  on.

  What students use it for:

  - Learning basic and advanced Windows 11 features through short training videos.
  - Getting familiar with Windows 11 if moving from Windows 10.
  - Accessing Microsoft 365 training resources.
  - Updating mapped drives in Windows if shared drive access changes.

  Common issues or things to know:

  - Windows 11 Quick Tips provides short videos focused on common Windows questions, interface changes, apps, and files.
  - Essential Training for Windows 11 is intended for users who are new to Windows or upgrading from Windows 10.
  - If you recently moved to Windows 11 and mapped drives or shared folders are not accessible, restart the computer first
  and check again.
  - If mapped drives still do not work, include your device name, campus location, whether you are on campus or remote, and
  which shared drive or folder you cannot access when you contact IT.
  - Some shared drives may appear differently after a Windows upgrade or migration.
  - Windows 10 reached Microsoft end of support on October 14, 2025.
  - If you are using a CCA-managed computer that is still on Windows 10, contact the CCA IT Helpdesk for current upgrade
  guidance.

  Where to get help / When to contact IT:

  - Contact the CCA IT Helpdesk if mapped drives, shared folders, printers, shortcuts, or other work resources do not work
  after moving to Windows 11.
  - Contact the CCA IT Helpdesk if you have questions, concerns, or problems with Windows 11 on a CCA-managed computer.

  CCA IT Helpdesk:
  - Phone: 303-360-4900, Monday–Friday, 8 a.m.–5 p.m.
  - After-hours/weekend support: 1-888-800-9198
  - Email: HelpdeskTickets.CCA@ccaurora.edu

  Video resources:

  - A video course is available for Windows 11 Quick Tips. It covers short, practical tips for common Windows 11 tasks
  and questions.
  - A video course is available for Essential Training for Windows 11. It covers how to use Windows 11 effectively,
  especially for users new to Windows or upgrading from Windows 10.
  - A video course is available for Microsoft 365 training. It covers getting started with Microsoft 365.
