D2L TROUBLESHOOTING

You are in the right place. These steps can help when D2L is blocking class access.

What this helps with:
- Use this guide when D2L (Brightspace) is not working as expected at the Community College of Aurora.
- It covers common sign-in, browser, course visibility, and assignment upload issues.

Common symptoms:
- D2L sign-in, course access, or assignment submission is not working as expected.
- A D2L page does not load correctly or shows an error.

Try this first:
1. Make sure you are using your correct CCA student account information.
2. Start the sign-in process again.
3. Enter your username and password carefully.
4. Keep the sign-in page open until the sign-in process finishes.
5. If D2L opens, go to the course, assignment, or page you need for class.

Cannot access D2L or log into D2L:
1. Check whether the problem is a password issue, account issue, or verification issue.
2. If the password is not working, use the CCA Login and Password Help guide.
3. If a verification prompt or code is blocking sign-in, use the MFA and Account Security guide.
4. If the sign-in page still fails, write down the exact error message before contacting IT.

Browser or cache issues:
1. Close the browser and open D2L again.
2. Try a different browser if the page does not load correctly.
3. If the problem continues, clear your browser cache and reopen D2L.

Assignment upload problems:
1. Open the assignment submission area in D2L and make sure you are submitting to the correct assignment.
2. Make sure you selected the correct file and gave the upload enough time to finish.
3. Refresh the page and try the upload again.
4. Confirm whether the problem affects one assignment or multiple uploads.

If that did not work:
- Check whether the issue is limited to sign-in, course access, or assignment submission.
- If you are still unable to submit an assignment, notify your instructor right away so they know you are having trouble.

Still need help?:
- Contact the CCA IT Helpdesk if you cannot sign in to D2L, your account access is not working, or D2L continues having technical problems after you retry sign-in, reopen the browser, or clear cache.

What to include when asking for help:
- Your student ID (S#) or CCA username if the issue affects sign-in.
- The exact error message or page behavior you see.
- The device and browser you are using.
- Whether the issue affects sign-in, a specific course, or assignment submission.
- Whether the issue affects one class or multiple classes.
- What steps you already tried.
- A screenshot if the page shows an error or failed submission.

Course not showing:
- Check whether you are looking in the correct term or course area.
- Some courses may not appear until the instructor makes them available.

When to contact your instructor:
- Contact your instructor if a course is missing, an assignment is unavailable, a due date looks wrong, or course content does not appear as expected.
